# 🎯 Project Gemini: 소비 카테고리 예측 파이프라인

## 1. 프로젝트 개요

이 프로젝트는 신용카드 거래 데이터를 분석하여, 사전에 정의된 6개의 비즈니스 중심 소비 카테고리로 분류하는 머신러닝 파이프라인입니다. 최종 목표는 사용자의 소비 패턴을 예측하여 맞춤형 광고 및 마케팅 활동(예: 쿠폰 발송)을 수행하는 것입니다.

### 주요 기술

*   **언어:** Python
*   **핵심 라이브러리:** Pandas, Scikit-learn, TensorFlow (Keras)
*   **실행 환경:** Python 가상 환경 (`mlcu`)
*   **특징:** GPU를 활용한 모델 학습 지원

### 아키텍처

프로젝트는 명확한 데이터 파이프라인 구조를 따릅니다.

1.  **데이터 수집 (`01_raw_data/`)**: 원본 신용카드 거래 데이터(`credit_card_transactions-ibm_v2.csv`)가 저장됩니다.
2.  **데이터 전처리 (`train_gpu.py`)**:
    *   `category_mapping.py`의 비즈니스 로직에 따라 MCC 코드를 6개의 주요 카테고리로 매핑합니다.
    *   정의된 6개 카테고리에 속하지 않는 데이터(약 620만 건)는 필터링하여 분석에서 제외합니다.
    *   'Amount'와 같은 데이터를 정제하고 'Is_Weekend'와 같은 파생 변수를 생성합니다.
    *   처리된 최종 데이터는 `02_processed_data/preprocessed_transactions_v2.csv`에 저장됩니다.
3.  **모델 학습 (`train_model.py`)**:
    *   전처리된 데이터를 로드하여 TensorFlow/Keras 신경망 모델을 구축하고 학습시킵니다.
    *   학습 과정은 TensorBoard를 통해 모니터링할 수 있습니다.
    *   학습이 완료된 모델은 `saved_models/` 디렉토리에 저장됩니다.

*   **주석**: 코드의 핵심 로직에는 한글 주석을 작성하여 가독성을 높입니다.
